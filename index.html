HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Semantic tags make it easier for pages to render
  • "touch index.html" is the terminal command to create new HTML page

CSS

  • touch style.css to create external css file from working branch
  • link element should have rel="stylesheet" and href="style.css"
  • classes in css are marked by a "." at the beginning of the selector
  • ID in css are marked by a "#" at the beginning of the selector
  • margin is space around outside element
  • padding is space around content inside element

Git

  • git status: checks what branch we are currently on
  • git branch: lists branches
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: takes us to main branch
  • git add -A: or git add . to commit files from working branch
  • git commit -m: "message"
  • git push origin + "branch name": Push branch onto main
  • git pull origin main: Pull base branch into feature branch
  • prework module 3.1.6 git flow
  • git remote add origin +URL to add remote URL to local repository

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • An array stores multiple items under a single variable name
  • Functions are not executed until called
